## 约定

- **文件后缀**：`.su`
- **入口**：含 `main` 的模块为可执行程序；无 `main` 的模块可作为库被 `import`
- **风格**：表达式为主；语句后加分号（如 `const`/`let`/`return`/`goto` 等）
