# 内置与标准库

## 概述

- **core**：无 OS 依赖的核心模块，用户通过 `import core.xxx` 引用。
- **std**：依赖 OS 的标准库，通过 `import std.xxx` 引用。
- 标准库扁平化：顶层为 `core/` 与 `std/`，无单独 `stdlib/` 目录；按需链接，不增大体积。

## core 模块（示例）

| 模块 | 说明 |
|------|------|
| `core.types` | 基础类型大小等（如 `size_of_i32()`、`size_of_bool()` 等占位/常量函数） |
| `core.builtin` | 内建/ intrinsics 占位，自举前最小可编译导出 |
| `core.option` | Option 类型（如 `Option_i32`、`none_i32`、`some_i32`、`unwrap_or_i32` 等） |
| `core.result` | Result 类型（如 `Result_i32`、`ok_i32`、`err_i32`、`unwrap_or_i32` 等） |
| `core.slice` | 切片相关（如 `len_i32(s: []i32): usize`） |
| `core.fmt` | 格式化占位（如 `fmt_i32`） |
| `core.mem` | 内存相关（如 `align_of_i32`） |
| `core.debug` | 调试（如 `assert(b: bool)`） |

## std 模块（示例）

| 模块 | 说明 |
|------|------|
| `std.io` | 输入输出 |
| `std.fs` | 文件系统 |
| `std.process` | 进程 |
| `std.path` | 路径 |
| `std.string` | 字符串 |
| `std.vec` / `std.vector` | 动态向量 |
| `std.map` | 映射 |
| `std.heap`、`std.runtime`、`std.error` 等 | 堆、运行时、错误等 |

具体 API 以各模块源码为准；自举前部分为占位实现。

## 内建函数（builtin）

- 当前 `core.builtin` 主要为占位，供 `import core.builtin` 通过编译。
- 编译器若识别特殊内建名，可映射到内部实现或 C 内建；具体内建列表以编译器实现为准。

## 使用方式

- 在入口或库文件顶部：`import core.types;`、`import std.io;` 等。
- 编译时通过 `-L` 指定库根目录（如项目根），使 `core.xxx`、`std.xxx` 能解析到对应 `.su` 模块。
